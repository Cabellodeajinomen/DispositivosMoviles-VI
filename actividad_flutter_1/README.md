# Aplicacion Personal en Flutter

Proyecto de practica para crear interfaces estaticas en Flutter.

## Objetivo

Construir tres pantallas visuales interconectadas con navegacion funcional:

1. Pantalla de inicio - con boton para ir al perfil
2. Pantalla de perfil personal - con botones para ir a inicio y hobbies
3. Pantalla de hobbies o intereses - con botones para navegar entre pantallas

## Estructura del proyecto

- lib/main.dart
- lib/app/app.dart
- lib/pantallas/pantalla_inicio.dart
- lib/pantallas/pantalla_perfil.dart
- lib/pantallas/pantalla_hobbies.dart

## Como ejecutar

1. Abrir el proyecto en VS Code o Android Studio.
2. Ejecutar este comando:

```bash
flutter run
```

## Como probar cada pantalla

La aplicacion permite navegar entre pantallas de forma sencilla:

- En la **Pantalla de Inicio**, presiona el boton "Ver mi perfil" para ir al perfil
- En la **Pantalla de Perfil**, usa los botones para navegar hacia Hobbies o volver a Inicio
- En la **Pantalla de Hobbies**, usa los botones para cambiar entre Inicio y Perfil
